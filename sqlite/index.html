sqlite

  • Dump data
  • Insert dump
  • In-Memory database

Dump data# 

mv sqlite.db sqlite.db.old 
sqlite3 sqlite.db.old .dump > backup.sql

Insert dump#

Keep in mind that backup.sql contains a CREATE-statement. If you want to just keep the data, delete the concerning lines.

# Linux
sqlite3 sqlite-new.db < backup.sql
# Windows
Get-Content backup.sql | sqlite3.exe sqlite.db

In-Memory database#

If you don’t want to create a file for the database, you can use the :memory: command. This way sqlite just creates a db in-memory, which gets delete when the process exists.

db, err := sql.Open("sqlite", ":memory:")
if err != nil {
    return err
}
